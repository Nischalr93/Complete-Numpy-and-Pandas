Define Python Pandas?¶

Pandas refers to software library explicitly written for python, which is used to analyze and manipulate data. Pandas is a open source library created by Wes Mckinney. It was released in 2008 and provided Data structure and operations to manipulate numerical and time-series data.

Pandas makes it very easy to perform machine learning operations on tabular data.

Pandas is a package for data manipulation and analysis in Python. The name Pandas is derived from the econometrics term Panel Data.

Why do you use Pandas?¶

Data representation Pandas provides extremely streamlined form of data representation. This helps to analyze and understand data bettery

Less Writing and more done: using pandas helps us to shorten the procedue of handling data, with time saved we can focus more on data analysis.

Pandas are really powerfull, they provide us with huge set of important commands and features which are used to easily analyze the data.

Efficiently handles large data Pandas help to save a lot of time by importing large amounts of data very fast.

Key features of Pandas:¶

Fast and efficient DataFrame object with default and customized indexing.**

Tools for loading data into memory data objects from different file formats**

Data alignment and handling of missing data**

Label based slicing, indexing and subsetting of large data sets.**

Columns from the data structure can be deleted or inserted.**

High performace merging and joining of data.**

What are the different types of Data structures in Pandas?¶

Pandas library support two major types of Data structures, DataFrame and Series.

Both these data structures are build on the top of Numpy.

Series is a one dimensional and simple Data structure.

While Pandas is two dimensional.

Another axis label is known as Panel is a 3 - dimensional data structure.

Explain Series in Pandas¶

Series is one dimensional array that can hold data values of any one data type(string, float, integer, python objects). 

Here the data axis label are called index.

Key Points

Homogeneous data
Size Immutable
Values of Data Mutable

Syntax for Series = pd.Series(Data, index, dtype, name, copy, fastpath)

Define Data frame?¶

A DataFrame if a 2-dimensional array with hetrogenous data in which data is aligned (represenated) in a tabular form with rows and columns.

Each column represents an attribute and each row represents a record.

Key Points

Heterogeneous data
Size Mutable
Data Mutable

Create a DataFrame using dict of ndarray / lists¶

All the ndarrays must be of the same length. If the index is passed, then the length of the index should be equal to the length of the array

If no index is passed, then by default index will be range(n), where n is the array length.
